# ویرایش‌ها


- افزودن پیکربندی موتور بازی‌سازی گودوت
- تغییر نام پوشه‌های کاربر:
	- `Documents` -> `documents`
	- `Downloads` -> `downloads`
	- `Music` -> `music`
	- `Pictures` -> `pictures`
	- `Public` -> `public`
	- `Templates` -> `templates`
	- `Videos` -> `videos`
- افزودن پیکربندی اف‌سیتک۵
- خودآغاز اف‌سیتک۵
- افزودن بریده‌های پرکاربرد خودم
- زین پس عبارت پوسته ` <> ` می‌باشد
